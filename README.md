## 📝 Описание программы

**Expense Chart** - это консольное приложение для учёта личных финансов с возможностью визуализации расходов. Программа позволяет:

- ✅ Добавлять, просматривать и удалять записи о расходах
- ✅ Фильтровать расходы по категориям и временным периодам
- ✅ Автоматически подсчитывать сумму расходов за выбранный период
- ✅ Строить наглядные графики распределения расходов по категориям (с использованием `matplotlib`)
- ✅ Сохранять и загружать данные в формате JSON
- ✅ Использовать наследование для разных типов расходов (еда с калориями, транспорт с расстоянием)

### Принципы ООП в программе:
- **Инкапсуляция**: все поля классов защищены (использование `_` и свойств)
- **Наследование**: классы `EdibleExpense` и `TransportExpense` наследуются от базового `Expense`
- **Полиморфизм**: переопределение методов `__str__` и `to_dict`
